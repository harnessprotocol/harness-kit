# Harness Kit

Harness Kit is the configuration console for all your AI coding harnesses.

Every AI coding tool has its own setup folder — Claude Code's `~/.claude`, Cursor's `.cursor`, Copilot's config. Different names, same shape, totally incompatible. **harness-kit** is the layer that turns all of them into one setup you carry with you: plugins, skills, MCP servers, hooks, and conventions, all declared in a single `harness.yaml`.

Configure once. Observe what your AI actually does. Share your setup with a teammate in one file.

## Quick install

```
/plugin marketplace add harnessprotocol/harness-kit
/plugin install research@harness-kit
```
